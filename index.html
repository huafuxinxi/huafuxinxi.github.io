开始使用 →

容易上手

克隆代码至本地，然后npm install下载依赖，执行命令npm run serve运行，npm run build打包上线 。

技术架构

享受 Vue + webpack 的开发体验，体验最新的TypeScript去编写高效严谨的代码，并向下兼容了JavaScript，使用Eslint规范代码，请严格遵守eslint规范。

灵活易用

如果你会使用HTML,JavaScript,CSS，那么相信你也会使用这个基础框架，在你通读完以后~~~

# 像数1，2，3一样简单

# 克隆代码至本地
git clone http://hfrdcgit.hfepay.local/kylin/kylin-web.git

# 安装依赖
npm install # 或者 cnpm install(确保安装了淘宝镜像)

# 线上开发
npm run serve

# 构建静态文件
npm run build # 自动生成对应静态文件夹和压缩文件
